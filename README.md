A Java-console based banking system built using Object-Oriented Programming(OOP) principles.
This projects demonstrate core java concepts such as methods,encapsulation,loops,conditionals,user input handling and basic transaction processing.
Features include create and manage a bank account,Deposit money,Withdraw Money,check account balance,Input validation for menu choices,confirmation before exiting the application,
Multiple transaction in one session,personalized banking messages and Encapsulation using getters and setters
Technology Used
Java
Notepad++
OOP Principles
Scanner class
